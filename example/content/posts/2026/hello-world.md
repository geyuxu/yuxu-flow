# Hello, World!

Welcome to your new Static Flow blog! This is a demo post to help you get started.

## Features

Static Flow supports many content formats:

- **Markdown** with syntax highlighting
- **Jupyter Notebooks** for technical content
- **LaTeX** documents (compiled to PDF)
- **Office documents** (.docx, .pptx)

## Code Example

Here's a simple Python example:

```python
def greet(name: str) -> str:
    """Return a greeting message."""
    return f"Hello, {name}!"

print(greet("World"))
```

## Math Support

You can write math equations using LaTeX syntax:

$$E = mc^2$$

## Getting Started

1. Edit `staticflow.config.yaml` to configure your site
2. Add your content to `content/posts/`
3. Run `staticflow build` to generate your site
4. Run `staticflow serve` to preview locally

Happy blogging!
